HTML-specific codes

Tag Use Description
HTML <html > Creates and HTML document
HEAD <head> Sets off the title & other info that isn't displayed
BODY <body> Sets off the visible portion of the document
B <b> Bold Words in HTML
H1-H6 <h1-h6> Creates headlines ex: h1, h2 etc.
TABLE <table> Used to create a Table
TD <td> Table Cells
TR <tr> Table Row
AUDIO <audio> Used to embed audio content
THEAD <thead> Used to section off the tables and columns
IMG <img> Adds an image
a href= <a href => Creates a hyperlink to various extentsions
STRONG <strong> Bolds a word
EM <em> Italicize a word
